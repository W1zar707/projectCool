Python
OpenCV
MediaPipe
Windows
HandMedia Controller — это забавный и полезный проект на Python, который позволяет управлять воспроизведением музыки/видео в любом плеере Windows (Spotify, YouTube, Яндекс.Музыка и т.д.) простыми жестами рук перед веб-камерой. Никаких клавиш, мыши или пульта — только вы и ваши руки!
Возможности
Левая рука — панель управления

Указательный палец ← Коснитесь большим пальцем → Предыдущий трек
Средний палец ← Коснитесь большим пальцем → Play / Pause
Безымянный палец ← Коснитесь большим пальцем → Следующий трек

На кончиках пальцев отображаются красивые иконки (←, ▶/Ⅱ, →), которые масштабируются под размер ваших пальцев.
Правая рука — информационная панель
На ладони правой руки в реальном времени показывается:

Название текущего трека
Имя исполнителя

Текст полупрозрачный и аккуратно центрируется в области ладони.
Как это выглядит
(Представьте здесь скриншоты или короткое видео: левая рука с иконками на пальцах, правая — с названием песни на ладони)
Требования

Windows 10/11
Python 3.9 или новее
Веб-камера
Установленные библиотеки:

Bashpip install opencv-python mediapipe pycaw pillow winsdk numpy
Также нужен файл модели MediaPipe: hand_landmarker.task (скачивается автоматически при первом запуске или вручную с официального сайта).
Установка и запуск

Склонируйте репозиторий или скачайте файлы:textmain.py
commands.py
finger.py
Положите иконки в папку с проектом:
previous.png
next.png
play.png
pause.png
(иконки должны иметь прозрачный фон — PNG с альфа-каналом)

(Опционально) Положите шрифт font.ttf в папку проекта для красивого текста на ладони.
Запустите:

Bashpython main.py

Поднесите руки к камере:
Левая рука — для управления
Правая рука — для отображения информации о треке


Технические детали

Распознавание рук: MediaPipe Hands (режим Live Stream)
Обработка видео: OpenCV
Управление плеером: Windows Media Control API (winsdk)
Наложение графики: Альфа-смешивание с поддержкой прозрачности
Масштабирование: Иконки и зоны касания автоматически подстраиваются под размер пальцев

Возможные доработки

Анимация нажатия кнопок
